# Lior Saban – Digital Business Card

A simple static landing page for GitHub Pages.

## Files
- `index.html` – landing page
- `styles.css` – styling
- `assets/profile-lior-saban.jpg` – portrait image
- `assets/og-lior-saban.jpg` – preview image for link sharing
- `assets/lior-saban.vcf` – downloadable contact card
- `.nojekyll` – disables Jekyll processing

## Setup
Upload all files to the root of your GitHub Pages repository, enable Pages, then generate a QR code pointing to the published URL.
